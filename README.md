# Superstore_Data_Analysis
A data analytics project built using Power BI to analyze retail sales data from the Superstore dataset. The dashboard provides insights into customer segmentation, sales performance, profit trends, and product analysis using interactive visuals, KPIs, and filters to support data-driven decision-making.


 # Full Project Overeview
 
📌 Project Overview

This project focuses on analyzing retail business data using the Superstore dataset to uncover meaningful insights related to sales performance, customer behavior, and product trends. The goal is to transform raw data into an interactive Power BI dashboard that helps stakeholders make informed decisions.

🎯 Objectives
Analyze overall sales, profit, and order trends
Identify high-value customers using segmentation techniques (RFM analysis)
Understand regional and category-wise performance
Detect loss-making areas and improvement opportunities
Build an industry-level interactive dashboard

🛠️ Tools & Technologies
Power BI (Data Visualization & Dashboarding)
Excel / CSV (Data Source)
Data Cleaning & Transformation (Power Query)
DAX (KPIs and calculated measures)

📂 Dataset
The dataset contains retail transaction data including:
Orders
Customers
Products
Sales & Profit
Regions and Categories

🔍 Key Features of Dashboard
1. 📈 Overview Dashboard
 Total Sales
 Total Orders
 Sales Trend Over Time
2. 👥 Customer Insights
  Customer Segmentation (RFM Analysis)
  Top Customers by Revenue
3. 📦 Product Analysis
  Sales by Category & Sub-category
  Top Performing Products
4. 🌍 Regional Analysis
  Sales by Region & State
  Geographic performance

Total Sales
Total Profit

⚙️ Data Processing Steps
Removed null and duplicate values
Standardized column formats
Created calculated columns (Profit %, RFM metrics)
Built relationships between tables
Designed interactive visuals with slicers 

📌 Insights Generated
Identified high-value customers contributing most revenue
Found categories with high sales but low profit
Discovered regional performance gaps
Highlighted opportunities for business growth

Business Implications
Focus on increasing Average Order Value (AOV) through bundling and cross-selling
Expand Home Office segment to diversify revenue streams
Strengthen customer retention programs for high-value customers
Invest in Technology category growth to sustain revenue leadership

Outcome

This project demonstrates strong skills in data analysis, dashboard design, and business intelligence. It helps in understanding customer behavior and improving strategic decision-making using data.
